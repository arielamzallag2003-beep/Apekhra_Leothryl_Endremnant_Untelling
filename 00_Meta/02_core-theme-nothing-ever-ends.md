# Core Theme -- Nothing Ever Truly Ends

## Énoncé central
Le thème fondamental de **Apekhra Leothryl -- the Endremnant Untelling** est le suivant :

> **Nothing ever truly ends.  
> Everything is always ending.**

Ce n'est ni une métaphore, ni un slogan narratif :  
c'est une **propriété structurelle de la cosmologie**.

---

## Clarification du paradoxe apparent
Les deux propositions ne se contredisent pas.

- **Nothing ever truly ends**  
  → Il n'existe presque rien qui puisse être annihilé de manière absolue.  
  Les âmes persistent. Les concepts persistent. Les récits persistent.  
  Même l'échec, la rupture ou la mort ne constituent pas une fin totale.

- **Everything is always ending**  
  → Toute chose est en tension constante vers sa propre cessation.  
  Les formes se dégradent. Les structures s'usent. Les récits se fissurent.  
  La stabilité est temporaire par nature.

La coexistence permanente de ces deux états est **le moteur du monde**, non une anomalie.

---

## Nature du thème
Ce thème n'implique pas :
- d'équilibre moral
- de cycle harmonieux
- de renaissance systématique
- de finalité cachée

Il décrit une condition où :
- la fin est un **processus**, pas un événement
- la persistance est un **résidu**, pas une victoire
- la continuité existe **sans légitimité absolue**

---

## Portée cosmologique
Ce thème s'exprime à tous les niveaux :

- **Tiers primordiaux** : rien n'est effacé sans recours à l'absolu (Null).  
- **Tiers structurels** : les systèmes fonctionnent tout en se dégradant.  
- **Concept-Entities** : les concepts tiennent sans jamais se fermer.  
- **Deities** : les autorités persistent sans être finales.  
- **Shadow** : l'échec devient une forme de survie.  
- **The Seer** : la clôture est empêchée par nécessité.

Aucun de ces éléments ne résout la tension ; ils l'entretiennent.

---

## Intention de ce thème
Ce thème sert à garantir que :

- la cosmologie reste ouverte
- les réponses définitives sont impossibles
- la fin du sens est évitée sans promettre de salut
- la narration n'aboutit jamais à un "état final"

Il n'existe pas de point où l'histoire est "terminée correctement".

---

## Limite volontaire
Ce fichier ne définit pas :
- comment ce thème est perçu par les civilisations
- comment il est exprimé mythiquement
- comment il est vécu individuellement

Ces aspects seront développés ailleurs.  
Ici, le thème est posé **comme fondation**, pas comme interprétation.
